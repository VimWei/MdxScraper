# 更新日志

## [未发布] - 现代化重构版本

### 新增功能
- **GUI 界面**: 全新的图形用户界面，基于 PySide6 开发
- **配置管理**: 智能的配置管理系统，支持配置方案保存和切换
- **模块化架构**: 将单文件脚本重构为模块化结构
- **类型注解**: 全面的类型注解支持，提高代码可维护性
- **测试覆盖**: 完整的单元测试和集成测试
- **文档系统**: 详细的 API 文档和用户指南

### 技术改进
- **依赖管理**: 使用 uv 进行现代化的包管理
- **配置格式**: 从 Python 配置文件迁移到 TOML 格式
- **错误处理**: 完善的异常处理和错误提示
- **性能优化**: 并发处理和缓存机制
- **跨平台支持**: 改进的跨平台兼容性

### 架构变更
- **目录结构**: 全新的项目目录结构
- **模块分离**: 核心功能拆分为独立模块
- **配置层次**: 三层配置管理（默认、用户、方案）
- **启动方式**: 保持便携式设计，支持多种启动方式
